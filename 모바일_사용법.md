# 📱 모바일에서 보는 방법

## 옵션 1 — GitHub Pages (무료, 가장 추천)

본인의 휴대폰/PC 어디서든 URL로 접속.

### 처음 한 번만 (5분)

1. **GitHub 계정** 준비 (없으면 https://github.com/join 가입)
2. 새 저장소 생성: 이름 예 `stock-viewer`, **Public** 으로
3. 컴퓨터에서 폴더 안의 다음 두 가지를 업로드:
   - `viewer.html`
   - `charts_forecast/` 폴더 통째로
4. 저장소 → **Settings** → **Pages**
   - Branch: `main`, Folder: `/ (root)` → Save
5. 1~2분 기다리면 `https://<사용자명>.github.io/stock-viewer/viewer.html` 활성화
6. 휴대폰 브라우저로 그 URL 접속 → 끝

### 매일 갱신 시
- `daily_run.bat` 돌아간 뒤 `viewer.html` + `charts_forecast/` 다시 GitHub에 업로드 (덮어쓰기)
- 또는 **GitHub Desktop** 쓰면 변경 파일 자동 감지 → 한 번 클릭

### 휴대폰 홈 화면에 추가
- iOS Safari: 공유 → "홈 화면에 추가"
- Android Chrome: 메뉴 → "홈 화면에 추가"
- → 앱처럼 아이콘으로 실행

## 옵션 2 — Netlify Drop (계정 불필요, 1분)

1. https://app.netlify.com/drop 접속
2. `viewer.html` + `charts_forecast/` 들어있는 폴더를 통째로 **드래그&드롭**
3. 즉시 `https://랜덤문자.netlify.app/viewer.html` URL 발급
4. 휴대폰에서 그 URL 접속

⚠️ 무료 한도 안에서 평생 호스팅. 단, URL이 무작위라 따로 메모해둬야 함.

---

## 옵션 3 — 로컬 파일 (인터넷 없이 쓰고 싶을 때)

### Android
1. PC와 USB 연결 → `viewer.html` + `charts_forecast/` 폴더를 폰 내부 저장소(예: `Download/stocks/`)로 복사
2. 폰에서 "내 파일" 앱으로 `viewer.html` 터치 → Chrome으로 열기
3. 끝

### iOS
1. iCloud Drive나 Files 앱에 `viewer.html` + `charts_forecast/` 폴더 업로드 (PC ⇄ iCloud)
2. iPhone Files 앱에서 `viewer.html` 터치 → 이미지 안 보일 가능성 있음 (iOS는 file:// 보안 제한)

→ iOS는 옵션 1(GitHub Pages) 강력 추천.

## 옵션 4 — 같은 Wi-Fi 안에서 PC를 임시 서버로

PC 켜져 있을 때만 접속 가능.

### PC에서 (한 줄)

```cmd
cd "C:\Users\정하림\Desktop\새 폴더 (2)"
py -m http.server 8000
```

### 휴대폰에서

같은 Wi-Fi 연결한 뒤, PC의 IP 확인 후 (`ipconfig`로 IPv4 주소, 예 `192.168.0.10`):

```
http://192.168.0.10:8000/viewer.html
```

브라우저에 입력 → 차트까지 다 동작.

## 어떤 옵션이 좋은가?

| 시나리오 | 추천 |
|---|---|
| 본인이 매일 휴대폰에서 보고 싶음 | **옵션 1 (GitHub Pages)** |
| 빨리 한번 보여주고 싶음 | **옵션 2 (Netlify Drop)** |
| Android, 인터넷 없이 | **옵션 3** |
| 가족/같은 사무실에서 잠깐 | **옵션 4** |

## 모바일에서 동작 확인된 것

- 종목 검색 (코드/이름)
- 시장·추천 필터
- 컬럼 헤더 터치로 정렬
- 행 터치 → 자동으로 차트 영역으로 스크롤
- 가로 스크롤 (테이블이 화면보다 넓을 때)
- 홈 화면 추가 시 앱처럼 동작
