# 한국주식 ML 분석 시스템 사용법

## 1. 폴더 구조

```
새 폴더 (2)/
├── data/
│   ├── tickers.xlsx              # 전 종목 리스트 (2,771)
│   ├── ohlcv/{코드}.parquet       # 종목별 일봉 (2010~)
│   ├── predictions_v2.xlsx        # 200R 학습 예측 결과
│   ├── predictions_pct.xlsx       # ★1000R 자체학습 — 예상 % 변동률 + 추천
│   ├── comparison.xlsx            # v1 vs v2 비교
│   ├── auto_ml_summary.csv        # 자체학습 요약 (AUC, RMSE 등)
│   ├── feature_importance_auto_*.csv
│   ├── training_log_v2.csv        # 학습곡선
│   ├── model_auto_cls.txt         # 분류 모델
│   └── model_auto_reg.txt         # 회귀 모델
├── charts/
│   └── {코드}.png                 # 종목별 전기간 차트 (MA + RSI + 예측)
├── download_krx.py                # 데이터 수집
├── train_predict.py               # 1차 학습 (200R)
├── make_charts.py                 # 차트 생성
├── auto_ml.py                     # ★1000R 자체학습 (분류 + 회귀)
└── predict_one.py                 # 종목 한 개 결과 조회
```

## 2. 어떤 파일을 봐야 하나?

### 가장 중요한 파일: `data/predictions_pct.xlsx`

전 종목에 대해 다음을 보여줍니다:

| 컬럼 | 의미 |
|---|---|
| 종목코드 / 종목명 / 시장 | KOSPI/KOSDAQ |
| 현재가 | 최근 영업일 종가 |
| 예상상승확률(%) | 분류 모델이 추정한 "20일 후 상승할 확률" |
| 예상변동률(%) | 회귀 모델이 추정한 "20일 후 % 변화" |
| 예상가격 | 현재가 × (1 + 예상변동률) |
| 추천 | ★강한매수 / 매수 / 관망 / 매도 / ★강한매도 |
| 신뢰도 | 두 모델 방향 일치 여부 + 강도 |

엑셀에서 `예상변동률(%)` 내림차순으로 정렬되어 있어, 위에서부터 보면 **상승 기대가 가장 큰 종목** 순서입니다.

## 3. 사용 시나리오

### 시나리오 A: 강한 매수 후보 찾기
1. `data/predictions_pct.xlsx` 열기
2. `추천` 컬럼에서 **★강한매수** 또는 **매수** 필터링
3. `신뢰도`가 **높음**인 것만 다시 필터링
4. 위에서부터 상위 N개 후보 확정

### 시나리오 B: 특정 종목 빠르게 보기
```bash
py predict_one.py 005930      # 코드로
py predict_one.py 삼성전자     # 이름으로
```
출력 예:
```
============================================================
  삼성전자 (005930, KOSPI)
  기준일       : 2026-04-28
  현재가       : 222,000원
  예상 상승확률: 47.3%
  예상 변동률  : -1.2%
  예상 가격    : 219,336원
  추천         : 관망
  신뢰도       : 중간
  차트         : charts/005930.png
============================================================
```
함께 표시된 `charts/005930.png`를 열면 시각적 확인 가능.

### 시나리오 C: 모델 성능 확인
- `data/auto_ml_summary.csv` — 분류 valid_auc, 회귀 spearman/rmse, 실제 학습 라운드 수
- `data/feature_importance_auto_reg.csv` — 어떤 지표가 % 예측에 가장 기여했는지

## 4. 재실행

| 작업 | 명령 |
|---|---|
| 데이터 갱신 (오늘까지) | `py download_krx.py` |
| 1000R 자체학습 다시 | `py auto_ml.py` |
| 종목별 차트 다시 그리기 | `py make_charts.py` |

## 5. 주의사항

- 모델 정확도는 valid AUC ~0.55 수준 — **완벽한 예측이 아니라 약한 통계적 우위**입니다.
- 단일 종목 예측보다 **상위 분위 종목 분산투자**가 통계적으로 안정적.
- 이 시스템은 학습/연구용이며 **투자 자문이 아닙니다**.
- 거시 충격(전쟁, 금리, 실적 발표 등)은 모델에 반영되지 않습니다.
